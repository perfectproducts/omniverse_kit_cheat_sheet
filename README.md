# Omniverse Kit Cheat Sheet — MkDocs + GitHub Actions

Mini-Doku & Snippet-Sammlung für Omniverse Kit. Baut automatisch mit **GitHub Actions** und läuft auf **GitHub Pages**.

## Quickstart
1. Dieses Paket in ein neues GitHub-Repo pushen (z. B. `omniverse-kit-cheat-sheet`).
2. In `mkdocs.yml` `repo_url` und `repo_name` anpassen.
3. In GitHub: **Settings → Pages** auf **GitHub Actions** stellen.
4. Push auf `main` —> Deployment erfolgt automatisch.

## Struktur
- `docs/cheatsheets/` — thematische Snippets und Kurzreferenzen
- `docs/treffen/` — Treffen mit Datum, Stichwort, Kurzbeschreibung & Aufnahme-Link
